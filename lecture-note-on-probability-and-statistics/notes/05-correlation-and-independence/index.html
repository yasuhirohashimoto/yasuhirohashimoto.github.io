相関と独立

確率変数が複数の場合

ノート「確率変数」では 1 つの確率変数 \(X\) とその分布を扱った．現実の問題では 2 つ以上の確率変数を同時に観察することが多い．たとえば，「ある学生の身長 \(X\) と体重 \(Y\)」，「あるユーザーの年齢 \(X\) とサイト滞在時間 \(Y\)」，「ある年の平均気温 \(X\) とアイスクリームの売り上げ \(Y\)」など．観測されるのはペア \((X_1, Y_1), (X_2, Y_2), \ldots, (X_n, Y_n)\) のように対になった標本である．

複数の確率変数を扱うと，自然と次の問いが立ち上がる：

  • \(X\) と \(Y\) はどう「関係」しているか？ \(X\) が大きいときに \(Y\) も大きい傾向があるか？
  • \(X\) の値を知ったときに \(Y\) の予測は変わるか？

本ノートでは以下の 2 つの概念を扱う：

  • 独立 \(\Rightarrow\) 無相関 ── 独立ならば必ず無相関．対偶をとると，相関があるなら独立ではない
  • 無相関 \(\nRightarrow\) 独立 ── 無相関だからといって独立とは限らない

両者は非対称な関係にあり，独立は無相関よりも強い概念 である．以下，まず相関を定義し，その幾何学的な意味を見たうえで独立との違いを明らかにする．

相関

2 つのデータ系列の直線的な関係の強さを表す指標として 相関 (correlation) がある．その前に，まず 2 つの量が同じ向きに動きやすいか，逆向きに動きやすいかを測る 共分散 (covariance) を導入しよう．

共分散と相関係数

分布側（母集団）では，2 つの確率変数 \(X, Y\) の共分散は

\[ \mathrm{Cov}[X,Y] \;=\; E\bigl[(X - E[X])(Y - E[Y])\bigr] \]

で定義される．この値は分布を特徴づける定数で，母共分散 (population covariance) と呼ばれ \(\sigma_{X,Y}\) とも表記する．ノート「母集団と標本」で見た \(E[X]\) と \(\mu_X\) の関係と同じで，\(\mathrm{Cov}[X,Y]\) は計算の記法，\(\sigma_{X,Y}\) はその値に与えた名前だ．

共分散の式変形や \(\mathrm{Cov}[X,Y]=E[XY]-E[X]E[Y]\) という計算しやすい形については，付録「確率変数の期待値と分散の式変形いろいろ」を参照．

一方，手元の \(n\) 個のデータ対 \((X_1, Y_1), \ldots, (X_n, Y_n)\) から同じ意味の量を計算したものが 標本共分散 (sample covariance) である．\(\bar{X},\bar{Y}\) をそれぞれの標本平均とすると，

\[ S_{X,Y} \;=\; \frac{1}{n} \sum_{i=1}^{n} (X_i - \bar{X})(Y_i - \bar{Y}) \]

で定義される．共分散が正なら \(X\) が平均より大きいときに \(Y\) も大きくなりやすく，負なら片方が大きいときに他方は小さくなりやすい．ただし共分散は \(X, Y\) の単位に依存するため，値そのものは比較しづらい．そこで，共分散を \(X, Y\) の標準偏差で割って \(-1\) から \(1\) の範囲に正規化したものが 相関係数 (correlation coefficient) だ．分布側の相関係数は

\[ \rho_{X,Y} \;=\; \frac{\mathrm{Cov}[X,Y]}{\sqrt{V[X]}\sqrt{V[Y]}} \;=\; \frac{\sigma_{X,Y}}{\sigma_X\sigma_Y} \]

で定義され，母相関係数 (population correlation coefficient) と呼ばれる．標本側の相関係数は，標本共分散と標本標準偏差 \(S_X, S_Y\)（標本分散 \(S_X^2, S_Y^2\) の平方根）を使って

\[ r_{X,Y} \;=\; \frac{S_{X,Y}}{S_X S_Y} \]

で定義され，標本相関係数 (sample correlation coefficient) と呼ばれる．標準偏差が 0 でない場合，共分散が \(0\) であることと相関係数が \(0\) であることは同じだ．このとき「\(X\) と \(Y\) は 無相関 (uncorrelated) である」という．

以降は，手元のデータ列を対象とした標本相関係数 \(r_{X,Y}\) をベースに話を進める．

ベクトルに見立てたデータ列の内積が 0

データ系列 \(X = (X_1, X_2, \ldots, X_n)\) は \(n\) 次元空間のベクトルと見なすことができる．そこで平均を引いた 中心化ベクトル を

\[ \widetilde{X} \;=\; (X_1 - \bar{X},\; X_2 - \bar{X},\; \ldots,\; X_n - \bar{X}) \]

と定めよう（\(Y\) 側も同様に \(\widetilde{Y}\) を定義する）．するとベクトルの内積は

\[ \langle \widetilde{X},\, \widetilde{Y} \rangle \;=\; \sum_{i=1}^{n} (X_i - \bar{X})(Y_i - \bar{Y}) \]

と書ける．これを共分散の定義式と見比べると，\(\langle \widetilde{X},\, \widetilde{Y} \rangle = n S_{X,Y}\) であることが分かる．すなわち，標本共分散は中心化ベクトルどうしの内積を \(n\) で割ったものに等しい．

さらに一歩進めて，中心化ベクトルを標準偏差で割った 標準化ベクトル \(\widetilde{X}/S_X,\ \widetilde{Y}/S_Y\) を考えよう．それぞれのノルムは \(\sqrt{n}\) なので※，ともにノルムで割って正規化した標準化ベクトルどうしの内積は

\[ \left\langle \frac{\widetilde{X}}{\sqrt{n}S_X},\ \frac{\widetilde{Y}}{\sqrt{n}S_Y} \right\rangle \;=\; \frac{1}{nS_XS_Y}\langle \widetilde{X},\ \widetilde{Y} \rangle \;=\; r_{X,Y} \]

となり，相関係数に等しい．つまり，2 つの標準化ベクトルのなす角を \(\theta\) とすると，相関係数は \(\cos\theta\) に等しい．相関係数が必ず \(-1\) から \(1\) の間に収まるのはこのためである．

  • \(r_{X,Y} = 1 \quad \rightarrow \quad\) 2 つのベクトルが同じ向き ── 完全な 正の相関 (positive correlation)
  • \(r_{X,Y} = 0 \quad \rightarrow \quad\) 2 つのベクトルが直交 ── 無相関 (uncorrelated, zero correlation)
  • \(r_{X,Y} = -1 \quad \rightarrow \quad\) 2 つのベクトルが逆向き ── 完全な 負の相関 (negative correlation)

言い換えると，無相関であることは，中心化したデータベクトルどうしが幾何学的に直交していること に他ならない．

標準化ベクトル \(\widetilde{X}/S_X\) のノルムを計算してみよう： \[ \left\|\frac{\widetilde{X}}{S_X}\right\| = \sqrt{\sum_{i=1}^{n} \left(\frac{\widetilde{X}_i}{S_X}\right)^2} = \frac{1}{S_X}\sqrt{\sum_{i=1}^{n} \widetilde{X}_i^2} = \frac{1}{S_X}\sqrt{nS_X^2} = \sqrt{n}. \]

独立

次に 独立 (independent) について考えよう．2 つの確率変数が独立な場合，片方の値が分かったとしても，どう頑張ってももう片方の値の予測精度を上げることはできない．ノート「ベイズの定理」では，データ（一方の事象）を得ることで仮説（もう一方の事象）の予測精度を上げられる ── つまり確率分布が変わる ── ことを見たが，そのような更新がそもそも起こらないのが「独立 」という状態である．

ノート「標本空間と事象」では，2 つの事象が同時に起こる確率を同時確率と呼んだ．確率変数を使う場合も同じで，「\(X=x\) かつ \(Y=y\)」となる確率を

\[ P(X = x, Y = y) \]

と表記する．離散型の場合，独立とは

\[ P(X = x \mid Y = y) \;=\; P(X = x) \]

が成り立つことである．この条件付き確率の式は「\(Y\) の値を知っても \(X\) の確率分布は変わらない」と読める．同じことを \(Y\) の側から見れば，「\(X\) の値を知っても \(Y\) の確率分布は変わらない」ともいえる．つまり，以下の 3 つの式

\[ \begin{aligned} P(X = x \mid Y = y) &= P(X = x),\\[6pt] P(Y = y \mid X = x) &= P(Y = y),\\[6pt] P(X = x, Y = y) &= P(X = x)P(Y = y) \end{aligned} \]

はすべて「\(X\) と \(Y\) が独立である」ことを示している．上の 2 つの式にそれぞれ \(P(Y = y)\) と \(P(X = x)\) をかけると 3 番目の式が導かれる．これは 同時確率がそれぞれの確率の積に等しい という形であり，2 つの確率変数が独立であることの典型的な表現である．これは 3 つ以上の確率変数が同時に独立である場合も同様で，たとえば \(n\) 個の確率変数 \(X_1, X_2, \ldots, X_n\) が同時に独立であることは

\[ P(X_1 = x_1, X_2 = x_2, \ldots, X_n = x_n) \;=\; P(X_1 = x_1)P(X_2 = x_2) \cdots P(X_n = x_n) \]

と表される※．

連続型の場合は，確率そのものではなく確率密度関数を用いて同様の条件を表す．同時確率密度関数や条件付き確率密度関数については，次のノート「同時確率と周辺確率」で導入する．

独立は「無相関」と似た概念だが，より強い条件である．2 つの確率変数が独立ならばそれらは必ず無相関だが，無相関だからといって独立とは限らない．両者の違いを，以下では具体例で見ていこう．

相関 0 でも独立ではない例

次の 4 組のデータを考えよう：

\(i\)1234
\(X_i\)\(-2\)\(-1\)\(1\)\(2\)
\(Y_i\)\(4\)\(1\)\(1\)\(4\)

これは \(Y_i = X_i^2\) という関係にあるデータである．\(Y\) は \(X\) から完全に決まっている（予測できる）ので明らかに \(X\) と \(Y\) は独立ではない．それでも相関は \(0\) になる．実際に計算してみよう．

まず標本平均を求めて

\[ \bar{X} \;=\; \frac{-2 - 1 + 1 + 2}{4} \;=\; 0, \qquad \bar{Y} \;=\; \frac{4 + 1 + 1 + 4}{4} \;=\; 2.5, \]

これらを用いて標本共分散を計算すると

\[ \begin{aligned} S_{X,Y} &= \frac{1}{4} \sum_{i=1}^{4} (X_i - 0)(Y_i - 2.5) \\ &= \frac{1}{4} \left[ (-2)(1.5) + (-1)(-1.5) + (1)(-1.5) + (2)(1.5) \right] \\ &= \frac{1}{4} \left[ -3 + 1.5 - 1.5 + 3 \right] \;=\; 0 \end{aligned} \]

となる．すなわち相関係数 \(r_{X,Y} = 0\) である．

\(X\) と \(Y\) のあいだには \(Y = X^2\) という明白な関係があるのに，相関はピタリと \(0\) になってしまう．相関は \(X\) そのものと \(Y\) そのものの直線的な関係（線形関係）しか捉えられない ことを示す例である．

視覚的な例 ── Datasaurus

もうひとつ有名な例として，Alberto Cairo が 2016 年に作成した Datasaurus※ を紹介しよう．これは散布図を描くと恐竜の輪郭になる 142 点からなる \((X,Y)\) のデータセットで，相関係数は \(r_{X,Y} \approx -0.06\) と「ほぼ無相関」である．しかし \(X\) を知れば \(Y\) の取りうる範囲は強く制約されるので，明らかに独立ではない．

複数の確率変数間の関係は，平均や分散，相関係数といった要約された統計量だけを見て判断するのではなく，散布図を描いて構造を確認することが重要である．

Alberto Cairo, "Download the Datasaurus: Never trust summary statistics alone; always visualize your data" (2016)．のちに Justin Matejka と George Fitzmaurice が同じ統計量をもつ 12 個の異なる形のデータセットを生成し，Datasaurus Dozen として発表している (Matejka & Fitzmaurice, CHI 2017)．

任意の関数で変換しても直交 ＝ 独立

では「独立」を幾何学的にどう表現できるだろうか？

鍵は \(X, Y\) をそのまま使うのではなく，任意の関数 \(f, g\) で変換した \(f(X), g(Y)\) についても常に無相関になるかどうか を問うことである．

先の \(n=4\) の例を用いて，\(f(X) = X^2,\ g(Y) = Y\) として新しいデータ系列の相関を計算してみよう．\(X^2\) の値は \(4, 1, 1, 4\) で平均は \(2.5\)，\(Y\) の値もまったく同じで \(4, 1, 1, 4\) で平均 \(2.5\) になるから，

\[ S_{f(X),g(Y)} \;=\; \frac{1}{4} \sum_{i=1}^{4} (X_i^2 - 2.5)(Y_i - 2.5) \;=\; \frac{1.5^2 + 1.5^2 + 1.5^2 + 1.5^2}{4} \;=\; 2.25 \]

と，今度は \(0\) ではなくなった．\(X\) そのものと \(Y\) そのものは相関 0 だが，\(X^2\) と \(Y\) は相関がある．これが「相関 \(0\) だが独立ではない」正体である．

一般に，「独立」とは次のように特徴づけられる：

\[ X \text{ と } Y \text{ が独立} \iff \text{任意の関数 } f, g \text{ について } f(X) \text{ と } g(Y) \text{ が無相関}^\text{※} \]
厳密には，ここでいう「任意の関数」とは期待値や共分散が発散してしまわないような十分広い関数クラスに限って考える．

まとめ

今回見た 2 つの概念を，ベクトルの言葉で並べると

  • 無相関 ── \(X\) と \(Y\) という 1 組 のベクトルが直交している．
  • 独立 ── \(f(X)\) と \(g(Y)\) という形で作れる あらゆる組 のベクトルが直交している．

独立とは，あらゆる関数変換を施しても直交性が保たれるという 無限個の直交条件の束 を意味する．この視点に立つと，「独立ならば無相関」はすぐに分かるし，逆が一般には成り立たないことも理解できるだろう．
