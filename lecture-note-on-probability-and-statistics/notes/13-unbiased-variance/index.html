母分散，標本分散，不偏分散

不偏推定量としての標本平均

ノート「大数の法則」で，サンプルサイズ \(n\) の標本平均 \(\bar{X}_n\) は母平均 \(\mu_X\) の 一致推定量 ── \(n\) を大きくすると確率的に真の値へ近づいていく推定量 ── であることを見た。実は標本平均にはもうひとつ嬉しい性質がある。有限の \(n\) のままでも，標本平均の期待値※はぴったり \(\mu_X\) と一致するのだ：

\[ \begin{aligned} E[\bar{X}_n] &= E\left[\frac{X_1 + X_2 + \cdots + X_n}{n}\right] \\[4pt] &= \frac{1}{n} \left\{E[X_1] + E[X_2] + \cdots + E[X_n]\right\} \\[4pt] &= \frac{1}{n} \left\{\mu_X + \mu_X + \cdots + \mu_X\right\} \\[4pt] &= \mu_X. \end{aligned} \]
標本平均の期待値については，付録「確率変数の期待値と分散の式変形いろいろ」も参照。

「推定量の期待値」が推定したい母数とぴったり一致するとき，その推定量を 不偏推定量 (unbiased estimator) と呼ぶ。一致推定量が \(n \to \infty\) での漸近的な性質だったのに対し，不偏推定量は 有限の \(n\) でも平均的には正しい ことを保証する性質だ。つまり標本平均は，一致推定量であると同時に不偏推定量でもある，二重にありがたい推定量なわけだ。

では，分散についてはどうだろうか。標本分散 \(S_{X,n}^2\) は，サンプルサイズを大きくすれば母分散 \(\sigma_X^2\) に確かに収束する ── つまり一致推定量である：

\[ \begin{aligned} S_{X,n}^2 &= \frac{1}{n}\sum_{i=1}^n (X_i-\bar{X}_n)^2 \\[4pt] &= \frac{1}{n}\sum_{i=1}^n \{(X_i-\mu_X)-(\bar{X}_n-\mu_X)\}^2 \\[4pt] &= \frac{1}{n}\sum_{i=1}^n (X_i-\mu_X)^2 -2(\bar{X}_n-\mu_X)\frac{1}{n}\sum_{i=1}^n (X_i-\mu_X) +(\bar{X}_n-\mu_X)^2 \\[4pt] &= \frac{1}{n}\sum_{i=1}^n (X_i-\mu_X)^2 -(\bar{X}_n-\mu_X)^2 \\[4pt] &\xrightarrow[n\to\infty]{} \sigma_X^2 - 0 = \sigma_X^2. && \text{※2 つの項にそれぞれ大数の法則を適用} \end{aligned} \]

しかし「有限の \(n\) で平均的に正しい」かどうか，つまり不偏推定量であるかどうかは別問題だ。本ノートでは結論として 標本分散は不偏推定量にはなっていない こと，すなわち \(E[S_{X,n}^2] \neq \sigma_X^2\) を示す。そのうえで，補正を施した不偏分散を導入する。

標本分散の期待値の計算

以下では添字を省略して，母平均 \(\mu\)，母分散 \(\sigma^2\) をもつ独立同分布な標本 \(X_1, \ldots, X_n\) について，サンプルサイズ \(n\) の標本平均を \(\bar{X}\)，標本分散を \(S^2\) と表記する。上で用いた式変形をそのまま用いて

\[ \begin{aligned} E[S^2] &= E\left[\frac{1}{n} \sum_{i=1}^n (X_i - \mu)^2 - (\bar{X} - \mu)^2\right] \\[4pt] &= \frac{1}{n} E\left[\sum_{i=1}^n (X_i - \mu)^2\right] - E\left[(\bar{X} - \mu)^2\right] \\[4pt] &= \frac{1}{n} E\left[\sum_{i=1}^n (X_i - \mu)^2\right] - V[\bar{X}]. \end{aligned} \]

右辺第一項を展開すると，

\[ \begin{aligned} \frac{1}{n} E\left[\sum_{i=1}^n (X_i - \mu)^2\right] &= \frac{1}{n} \left\{E[(X_1 - \mu)^2] + E[(X_2 - \mu)^2] + \cdots + E[(X_n - \mu)^2]\right\} \\[4pt] &= \frac{1}{n} \left\{V[X_1] + V[X_2] + \cdots + V[X_n]\right\} \\[4pt] &= \frac{1}{n} \cdot n \sigma^2 = \sigma^2. \end{aligned} \]

右辺第二項は，標本平均の分散※として

\[ \begin{aligned} V[\bar{X}] &= V\left[\frac{X_1 + X_2 + \cdots + X_n}{n}\right] \\[4pt] &= \frac{1}{n^2} \left\{V[X_1] + V[X_2] + \cdots + V[X_n]\right\} \\[4pt] &= \frac{1}{n^2} \cdot n \sigma^2 = \frac{\sigma^2}{n}. \end{aligned} \]
標本平均の分散については，付録「確率変数の期待値と分散の式変形いろいろ」も参照。

以上から，

\[ E[S^2] = \sigma^2 - \frac{\sigma^2}{n} = \frac{n - 1}{n} \sigma^2. \]

つまり 標本分散の期待値は母分散に一致せず，不偏推定量になっていない。具体的には \(\frac{n-1}{n}\) 倍に系統的に小さく評価されてしまう。

不偏分散

この「標本分散は平均的に母分散の \(\frac{n-1}{n}\) 倍になる」という結果を逆手にとると，標本分散を \(\frac{n}{n-1}\) 倍した値 ── つまり分散を計算する際に \(n\) ではなく \(n - 1\) で割った値

\[ s^2 = \frac{n}{n-1} S^2 = \frac{1}{n - 1} \sum_{i=1}^n (X_i - \bar{X})^2 \]

を使えば，

\[ \begin{aligned} E[s^2] &= E\left[\frac{n}{n-1} S^2\right] = \frac{n}{n-1} E[S^2] \\[4pt] &= \frac{n}{n-1} \cdot \frac{n-1}{n} \sigma^2 = \sigma^2 \end{aligned} \]

のように期待値を母分散に一致させることができる。このように補正した分散が 不偏分散 (unbiased variance) である。※

大文字 \(S^2\) を標本分散，小文字 \(s^2\) を不偏分散と使い分けている点に注意。文献によっては記号や呼び方が違うこともあるので，前後の文脈で判断してほしい。

直感的な説明

母分散は，真の平均 \(\mu\) を中心にした二乗偏差の期待値 \(E[(X-\mu)^2]\) である。一方，標本分散では，目の前の標本に合わせて動く標本平均 \(\bar{X}\) を中心に平均二乗偏差 \(\frac{1}{n}\sum_{i=1}^n (X_i-\bar{X})^2\) を計算する。標本平均はその標本に対する平均二乗偏差を最小にする中心なので※，\(\bar{X}\) を中心に測った平均二乗偏差は \(\mu\) を中心にした場合よりも小さくなりやすい。これが 標本分散の期待値が母分散よりも小さくなる 理由である。

任意の中心 \(a\) について，次の分解が成り立つ： \[ \frac{1}{n}\sum_{i=1}^n (X_i-a)^2 = \frac{1}{n}\sum_{i=1}^n (X_i-\bar{X})^2 +(\bar{X}-a)^2. \] 右辺第二項は \(0\) 以上なので，二乗偏差は \(a=\bar{X}\) のとき最小になる。つまり標本平均は，その標本に対してもっとも都合のよい中心である。そのため，\(\bar{X}\) を中心に測る標本分散は，\(\mu\) を中心に測る場合より小さく出やすい。

以下のデモでは，母集団からサンプルを抽出して標本分散と不偏分散を計算する操作を繰り返すことで，標本分散と不偏分散の期待値の違いが体感できる。抽出を重ねるごとに，標本分散の平均が母分散より小さい値に落ち着いていく様子と，不偏分散の平均が母分散に近づいていく様子が見えるはずだ。

たまにある間違い

「分散を計算するときには \(n\) ではなく \(n - 1\) で割れば偏りがないんだ！」と機械的に覚えて，どんな場面でも \(n - 1\) で割ろうとするのは誤りである。使い分けが必要だ：

  • 今，目の前にあるサンプルデータがデータのすべてである場合，つまり 全数調査 (complete enumeration) の場合，あるいは手元のデータそのもののばらつきを記述したい場合は，分散の定義として当然 \(n\) で割るのが正しい。たとえばクラスの中でテストの点数の分散を求めて，そこから個々人の偏差値を計算したいときなどはこれに該当する。
  • 一方，目の前にあるサンプルデータが巨大な母集団から抜き出された一部であり，そこから母集団全体の分散を 推定 したい場合は \(n - 1\) で割るのが正しい（平均的に偏りのない推定量になる）。たとえば同じテストについて，全国の学生の分散を，あるクラスの学生の点数から推定したい場合などはこれに該当する。

つまり，標本分散を使うべきか不偏分散を使うべきかは 分析の目的に依存する。※テキストによっては単に「分散」と書いて不偏分散を指している場合があるので，前後の文脈に注意しよう。

「\(n\) が大きければ \(n\) で割ろうが \(n - 1\) で割ろうが大して変わらないから，別に気にしなくてよいのでは？」という考え方もあるだろう。しかし \(n\) で割る操作は記述統計，\(n - 1\) で割る操作は統計的推論を想定しており，両者の態度には本質的な違いがある。
