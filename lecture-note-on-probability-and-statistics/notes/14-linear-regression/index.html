線形回帰

「勉強時間と試験の得点」「年齢と血圧」「広告予算と売上」「SNS 利用時間と幸福度」── 2 つの変数の間にゆるい関係がありそうなとき，「両者の関係を定量的に捉えられないだろうか？」と問うのは自然な発想だ。これを系統的に行う手法が 線形回帰 (linear regression) である。※本ノートでは，1 つの変数からもう 1 つの変数を予測する最も基本的な形 ── 単回帰 (simple linear regression) ── に絞って，その関係性を推定する方法を見ていく。

「回帰」（＝元に戻る）という呼び方には最初戸惑いを覚えるだろう。由来は，19 世紀の英国で Francis Galton が親の身長から子の身長を予測する直線 ── まさに本ノートで学ぶ回帰直線 ── を当てはめたとき，「非常に高身長の親の子は，平均よりは高いが親ほどは高くない」，つまり予測が平均の方へ寄る傾向を見つけ，これを 平均への回帰 (regression to the mean) と呼んだことにある。この直線の予測が必ず「平均へ戻る」理由については「相関係数との関係」の節で明らかにする。

説明変数と目的変数

線形回帰では，扱う 2 つの変数のうち予測したい方を 目的変数 (response variable; 被説明変数) と呼び，予測のもとにする方を 説明変数 (explanatory variable; 独立変数) と呼ぶ。たとえば「勉強時間 \(\to\) 試験の得点」を考えるなら，得点が目的変数で勉強時間が説明変数。※

「試験の得点 \(\to\) 勉強時間」としても構わない。得点から勉強時間を推定したい場合もあるだろう。この場合は，得点が説明変数で勉強時間が目的変数となる。

慣習として説明変数を \(x\)，目的変数を \(y\) で書くことが多い。※ \(n\) 個の標本 \((x_1, y_1), (x_2, y_2), \ldots, (x_n, y_n)\) を散布図にプロットしたとき，その「全体の傾向」を直線で近似できれば，新しい \(x\) の値からそれに対応する \(y\) の値を予測できる ── これが線形回帰の最終ゴールだ。

これまで標本を確率変数として \(X_i\) など大文字で書いてきたが，本ノートでは 実際に観測された値（実現値，観測値）として扱う ため小文字 \(x_i\) を使う。確率分布から「これからサンプリングされる量」ではなく，「すでに手元にある量」として議論する場面と思えばよい。

線形モデルと残差

そもそも，データを 何らかの関数で近似する 統計手法を 回帰 (regression) と総称する。どのような関数を仮定するかによって分類され，本ノートが扱う線形回帰はその名のとおり 直線（1 次関数） を使うケースだ。曲線で近似するなら多項式回帰や非線形回帰になる。非常に柔軟な関数の形をもつ ニューラルネットワーク (neural network; NN) など，現代の 機械学習 (machine learning; ML) の手法の多くも，本質的には「関数近似」の枠組みで理解できる。

直線で近似したいとき，仮定するモデルは

\[ y = a x + b \]

の形である。 \(a\) を 傾き (slope)，\(b\) を 切片 (intercept) と呼び，まとめて 回帰係数 (regression coefficient) と総称する。回帰係数は，直線の形を決める母数である。観測値 (observed value; 実現値) \(x_i\) に対し，この直線が予測する \(y\) の値を 予測値 (predicted value) と呼んで

\[ \hat{y}_i = a x_i + b \]

と書く。もちろん観測値 \(y_i\) が予測値 \(\hat{y}_i\) にぴったり乗ることはまずない。両者のずれ

\[ e_i = y_i - \hat{y}_i = y_i - (a x_i + b) \]

を 残差 (residual) と呼ぶ。データ全体について残差が小さく抑えられるほど，直線がデータをよく近似しているといえる。

最小二乗法

問題はもちろん「\(a\) と \(b\) をどう決めるか」だ。方針は「残差を全部足し合わせた量が最小になるよう選ぶ」こと。ただし単に「全部足す」だと，正の残差と負の残差が打ち消し合い，当てはまりが悪くても和は 0 になりうる ── たとえば重心を通る直線は傾きによらず残差の和が 0 になる。※そこで二乗してから足し合わせる：

\[ \mathrm{RSS}(a, b) = \sum_{i=1}^n e_i^2 = \sum_{i=1}^n (y_i - \hat{y}_i)^2 = \sum_{i=1}^n (y_i - a x_i - b)^2. \]
「絶対値の和でもよさそうだが」と思った人へ。確かに \(\sum |e_i|\) を最小化する方法もある。これは 最小絶対偏差法 (least absolute deviations; LAD) と呼ばれ，外れ値に強いことで知られている。ただし絶対値関数は折れ点で微分できないので解析的な扱いが面倒なのが難点。二乗にすると微分しやすく，しかも残差が大きい点ほど強くペナルティをかけられるという利点がある。

この 残差二乗和 (residual sum of squares; RSS) を最小化する \(a, b\) を求める手法を 最小二乗法 (ordinary least squares; OLS) と呼ぶ。

最小二乗解の導出

\(\mathrm{RSS}(a, b)\) を最小化する \((a, b)\) は，\(a\) と \(b\) それぞれで偏微分してゼロになる連立方程式を解けばよい。 \(\mathrm{RSS}\) は \(a, b\) について下に凸の二次関数なので，傾きが 0 になる点で最小になる。

\[ \begin{aligned} \frac{\partial \mathrm{RSS}}{\partial a} &= -2 \sum_{i=1}^n x_i (y_i - a x_i - b) = 0, \\[4pt] \frac{\partial \mathrm{RSS}}{\partial b} &= -2 \sum_{i=1}^n (y_i - a x_i - b) = 0. \end{aligned} \]

これを解くと（途中の式変形は付録「線形回帰における回帰係数の計算と式変形」を参照）：

\[ \hat{a} = \frac{S_{X,Y}}{S_X^2}, \qquad \hat{b} = \bar{y} - \hat{a}\, \bar{x}. \]

ここで \(S_{X,Y}\) は標本共分散，\(S_X^2\) は標本分散である（ノート「相関と独立」参照）。また，\(a, b\) ではなく \(\hat{a}, \hat{b}\) と「ハット」を付けているのは，真の値 \(a, b\) と区別するためである。真の \(a, b\) は通常は直接観測できる量ではなく，データから求められるのは，あくまでそれらの 推定値 (estimate) に過ぎない。ハット記号は，「真値そのものではなく，データから推定された値」であることを表す慣習である。※

回帰直線の予測値 \(\hat{y}_i\) にもハットを付けたのは，同様に真の値と区別するためである。ただし，\(\hat{y}_i\) は観測された \(x_i\) から算出される値であり，\(\hat{a}, \hat{b}\) は未知の母数をデータから推定した値であるため，前者を 予測値，後者を 推定値 と呼び分ける。

式 \(\hat{b} = \bar{y} - \hat{a}\, \bar{x}\) を書き直せば \(\bar{y} = \hat{a}\, \bar{x} + \hat{b}\) ── つまり 最小二乗解の直線は必ずデータの重心 \((\bar{x}, \bar{y})\) を通る。データ全体に対する誤差を最小化する直線が重心を通るというのは，直感的にも納得しやすい性質だ。

また，最小二乗解を代入したときの \(\mathrm{RSS}\) も書いておこう。 \(\mathrm{RSS}\) の式に \(\hat{a}=S_{X,Y}/S_X^2\) と \(\hat{b} = \bar{y} - \hat{a}\bar{x}\) を代入して，

\[ \begin{aligned} \mathrm{RSS} &= \sum_{i=1}^n \left\{(y_i - \bar{y}) - \frac{S_{X,Y}}{S_X^2}(x_i - \bar{x})\right\}^2 \\[4pt] &= n\left(S_Y^2 - \frac{S_{X,Y}^2}{S_X^2}\right). \end{aligned} \]

この式は次の節で使う。

相関係数との関係

ノート「相関と独立」で見た相関係数 \(r_{X,Y} = S_{X,Y} / (S_X S_Y)\) を使うと，傾き \(\hat{a}\) は

\[ \hat{a} = \frac{S_{X,Y}}{S_X^2} = \frac{S_{X,Y}}{S_X S_Y} \cdot \frac{S_Y}{S_X} = r_{X,Y} \cdot \frac{S_Y}{S_X} \]

と書ける。つまり傾きは「相関係数 × \(y\) のばらつき / \(x\) のばらつき」の形をしている。特に \(\hat{a}\) の符号は \(r_{X,Y}\) の符号と完全に一致する ── 正の相関なら正の傾き，負の相関なら負の傾き。

逆向きに見ると，両軸をそれぞれ標準化（平均を引いて標準偏差で割る）した散布図 での回帰直線の傾きが相関係数 \(r_{X,Y}\) そのものになる。相関係数と単回帰の傾きは，スケールを揃えるかどうかだけが違う，表裏の関係にあるわけだ。

この標準化した傾きが，冒頭の「回帰」という名前の種明かしになっている。標準化した散布図では，\(x\) が平均から標準偏差 \(k\) 個分離れていても，予測値 \(\hat{y}\) は平均から \(|r_{X,Y}| \cdot k\) 個分しか離れない。相関が完全（\(\pm 1\)）でない限り，回帰直線の予測はつねに入力のズレより平均寄りになる ── Galton が親子の身長に見た「平均への回帰」は，この \(|r_{X,Y}| < 1\) の現れである。※

「平均への回帰」は，何かを平均へ引き戻す力がはたらいていることを意味しない。逆向きに子の身長から親の身長を予測しても，同じように平均へ回帰する。相関が完全でないことの統計的な帰結にすぎず，これを実在の力と取り違える誤りには 回帰の誤謬 (regression fallacy) という名前までついている。

この関係は，回帰直線による予測のよさともつながっている。まず基準として，「どんな \(x\) が来ても \(y\) の平均値 \(\bar{y}\) で予測する」という素朴な方法を考えよう。このときの二乗誤差の合計は

\[ \mathrm{TSS} = \sum_{i=1}^n (y_i - \bar{y})^2 \]

である。これを 総平方和 (total sum of squares; TSS) と呼び，「平均値だけでは説明できずに残っている \(y\) のばらつき」と解釈できる。一方，回帰直線 \(\hat{y}_i = \hat{a}x_i + \hat{b}\) で予測したときの二乗誤差の合計は

\[ \mathrm{RSS} = \sum_{i=1}^n (y_i - \hat{y}_i)^2. \]

つまり，回帰直線によって減った二乗誤差は \(\mathrm{TSS} - \mathrm{RSS}\) である。この減った分が，もとの誤差 \(\mathrm{TSS}\) の何割にあたるかを表したものが 決定係数 (coefficient of determination) \(R^2\) である：

\[ \begin{aligned} R^2 &= \frac{\mathrm{TSS} - \mathrm{RSS}}{\mathrm{TSS}} \\[4pt] &= 1 - \frac{\mathrm{RSS}}{\mathrm{TSS}} \\[4pt] &= 1 - \frac{n\left(S_Y^2 - \frac{S_{X,Y}^2}{S_X^2}\right)}{nS_Y^2} \\[4pt] &= \frac{S_{X,Y}^2}{S_X^2 S_Y^2} \\[4pt] &= r_{X,Y}^2. \end{aligned} \]

このように，説明変数が 1 つだけの単回帰では 決定係数が \(R^2 = r_{X,Y}^2\) と一致する。つまり，相関係数の二乗は，直線による予測が平均値予測からどれだけ改善したかの目安にもなっている。たとえば \(R^2 = 0.64\) （相関係数 0.8）なら，このデータ上では，平均値だけで予測する場合に比べて，二乗誤差を 64% 減らせたという読み方ができる。

まとめ

線形回帰モデル \(y = ax + b\) は単純なモデルだが，線形近似 (linear approximation) は数学的に非常に強力な考え方である。実際，微分可能な関数は局所的には直線で近似できる（Taylor 展開の 1 次近似）。複雑な関係であっても，「まず直線で近似してみる」という発想が広く使われる理由のひとつはここにある。

さらに，①関数の形を仮定し，②データからそのパラメータを推定し，③新しい入力に対する予測を行う，という流れは，機械学習全般に共通する基本構造でもある。直線を使えば線形回帰，多項式を使えば多項式回帰，さらにカーネル法・決定木・ニューラルネットワークへと発展させていけば，現代の機械学習の主要手法につながっていく。その意味で，線形回帰は機械学習の最も基本的な入り口のひとつ と言ってよい。

最後に，線形回帰を使うときの注意点を押さえておこう：

  • そもそも関係が直線でない：薬の用量と効果のように，少なすぎれば効かず多すぎれば毒になる ── そんな逆 U 字型の関係に無理に直線を当てはめても正しい予測は得られない。注目するデータの範囲内では直線でも，その範囲外でも続いてくれる保証はないので，外挿 (extrapolation) は控えめにしておくのが無難だろう。※
  • 相関は因果ではない：よい当てはまりが得られても，\(x\) が \(y\) の原因とは限らない。アイスクリームの売上と水難事故件数は強く正の相関を示すが，もちろんアイスを売ったから事故が起こったわけではなく，どちらも気温という共通因子 ── 交絡変数 (confounding variable) に影響されているだけである。
「カントリーマアムはステルス値上げで年々小さくなっているので，このペースで縮み続けるといつか大きさが負になってしまう」という冗談があるが，もちろんサイズが \(0\) 未満になることはない。（公式には，実は重量変更は過去 2 回しか行われていないという情報もある。）

発展的な内容

最小二乗法は実はすでに学んだ最尤法の一種である。線形回帰を確率モデルとして見ると，観測値 \(y_i\) は

\[ Y_i = a x_i + b + \varepsilon_i, \qquad \varepsilon_i \sim \mathcal{N}(0, \sigma^2) \]

の実現値だと考えられる。ここで \(\varepsilon_i\) は，直線だけでは説明しきれないずれをモデルの側にあらかじめ組み込んだランダムな 誤差項 (error term) である。線形回帰で登場した 残差 と似た形をしているが，残差は推定した直線モデルと観測データの誤差を表す量である。一方，誤差項は「観測データが真のモデルからどれだけ外れるか」を表す量である。

この仮定のもとで，与えられた \(x_i\) と母数 \((a, b, \sigma)\) に対して \(y_i\) が観測される確率密度は

\[ f(y_i; x_i, a, b, \sigma) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left( -\frac{(y_i - a x_i - b)^2}{2\sigma^2} \right) \]

となる。確率変数 \(\varepsilon_i\) が平均 \(0\)，分散 \(\sigma^2\) の正規分布に従うという仮定から導かれる式だ。いま独立な \(n\) 個のデータ点が観測されたならば，全体の尤度はそれらの積（ノート「統計的推定：最尤法の初歩」の手順 2 と同じ要領）

\[ L(a, b, \sigma) = \prod_{i=1}^n f(y_i; x_i, a, b, \sigma) \]

で表される。対数尤度をとると，積は和に変わって：

\[ \log L(a, b, \sigma) = -\frac{n}{2} \log (2\pi\sigma^2) - \frac{1}{2\sigma^2} \sum_{i=1}^n (y_i - a x_i - b)^2 = -\frac{n}{2} \log (2\pi\sigma^2) - \frac{\mathrm{RSS}(a, b)}{2\sigma^2} \]

を得る。ここでは，誤差のばらつき \(\sigma\) はひとまず固定されているものとして，「どの \(a, b\) が観測データをもっともよく説明するか」だけに注目しよう。このとき，第一項は \((a, b)\) に依存しないので無視できる。残るのは \(-\mathrm{RSS}(a, b) / (2\sigma^2)\) で，マイナス符号と正の定数 \(1/(2\sigma^2)\) を考えれば，\(\log L\) を最大化することは \(\mathrm{RSS}(a, b)\) を最小化することと完全に等しい。つまり最小二乗解 \(\hat{a}, \hat{b}\) は最尤推定値そのものだ。
