無相関だが独立ではない 2 つの確率変数の周辺確率

ノート「同時確率と周辺確率」のデモでは，共分散を \(0\) にすると条件付き分布が周辺分布に一致し，\(X\) と \(Y\) が独立になった。あのデモの脚注で触れたように，これは二変量正規分布という特別な分布を使っていたからで，一般には 共分散が \(0\)（無相関）でも独立とはかぎらない。本付録では，その反例を同じ種類のデモで体感する。

円周上に散らばる標本点

標本点を，原点を中心とする半径 \(1\) の円周のまわりに散らばらせてみよう。角度 \(\theta\) は \(0\) から \(2\pi\) まで一様に選び，中心からの距離 \(r\) は平均 \(1\)，標準偏差 \(\sigma_r\) でばらつくものとする。この \((r, \theta)\) から，直交座標を

\[ X = r\cos\theta, \qquad Y = r\sin\theta \]

で定める。\(\sigma_r = 0\) なら，すべての点はちょうど単位円周上に載る。\(\sigma_r\) を大きくするほど，円周は太いドーナツ状ににじんでいく。

まず，無相関であること

角度 \(\theta\) が一様なので \(E[\cos\theta] = E[\sin\theta] = 0\)，したがって \(E[X] = E[r]\,E[\cos\theta] = 0\) となる（\(Y\) も同様に \(E[Y] = 0\)）。共分散を計算すると，

\[ \begin{aligned} \mathrm{Cov}(X, Y) &= E[XY] - E[X]\,E[Y] \\[4pt] &= E[\,r^2\cos\theta\sin\theta\,] && \text{※} E[X] = E[Y] = 0 \\[4pt] &= E[r^2]\,E[\cos\theta\sin\theta] && \text{※} r \text{ と } \theta \text{ は独立} \\[4pt] &= E[r^2] \cdot 0 = 0. \end{aligned} \]

最後の \(E[\cos\theta\sin\theta] = \tfrac{1}{2}E[\sin 2\theta] = 0\) は，\(\sin 2\theta\) を一周ぶん平均すれば消えることから分かる。というわけで，この \(X\) と \(Y\) は共分散が \(0\) ── 相関係数も \(0\)，つまり 無相関 である。

しかし，独立ではないこと

では独立だろうか？ そうではない。\(\sigma_r = 0\) の場合を考えると，\(X = x\) と分かった瞬間に \(Y\) の値は \(\pm\sqrt{1 - x^2}\) の 2 つに限られてしまう。※ \(X\) の値を知ると \(Y\) について分かることが変わるのだから，条件付き分布は周辺分布と等しくならない ── \(f_{Y \mid X}(y \mid x) \neq f_Y(y)\) ── のであり，\(X\) と \(Y\) は独立ではない。無相関は「直線的な関係の強さ」しか測らないので，円周上のように曲がった依存関係は共分散にはまったく現れないのである。

\(\sigma_r > 0\) のときも事情は同じで，\(X = x\) を固定するごとに \(Y\) の散らばり方（分布の形）が変わる。\(\sigma_r = 0\) は，それが \(2\) 点に退化した極端な場合というだけだ。

動かして体感しよう

下の散布図は，上のルールで生成した \(N\) 個の標本点 \((X_i, Y_i)\) を描いたものである。操作はノート「同時確率と周辺確率」のデモと同じで，上側と右側の青いヒストグラムが周辺分布 \(f_X, f_Y\)，等高線が同時密度を表す。ヒストグラムにポインタを合わせる（または図にフォーカスして矢印キーを押す）と，その区間に入る点だけを集めた条件付き分布が橙で重なる。

円周上に散らばる標本点を描画する領域（WebGL 2 が必要）

\(\sigma_r\) を \(0\) に近づけると，標本点は細い円周に収束する。このとき上側のヒストグラムで中央（\(x \approx 0\)）のビンを選ぶと，右側の橙のヒストグラムは \(y \approx +1\) と \(y \approx -1\) の 2 か所に山をもつ ── 青い周辺分布 \(f_Y\) とはまるで違う形だ。いっぽう端（\(x \approx 1\)）のビンを選ぶと，橙は \(y \approx 0\) の 1 か所に集まる。選ぶ \(x\) によって条件付き分布の形ががらりと変わること，これが \(X\) と \(Y\) が独立でない証拠である。それでも，散布図の上下左右の対称性ゆえに共分散はゼロのままなのだ。

ちなみに \(\sigma_r = 0\) では，周辺分布 \(f_X, f_Y\) は両端（\(\pm 1\)）で高くなる U 字型になる（逆正弦分布と呼ばれる）。円周を真上から見下ろすと，左右の端のあたりで点が密に重なって見えることに対応している。
