モーメント母関数

モーメント母関数 (moment generating function) とは，\(E[e^{tX}]\) で定義される関数である。 \(X\) について期待値をとるので \(X\) は消え，\(t\) についての関数となる。 \(t\) って何？と思うかもしれないが，0 に近い，後で消えてもらう仮の変数程度に思っておこう。確率変数 \(X\) についてのモーメント母関数を，ここでは \(M_X(t)\) と表記する。

モーメントを生み出す関数

いま，期待値の中の指数関数を，付録「指数関数の定義」で見たような無限に続く和に展開してみよう。※

\[ \begin{aligned} M_X(t) = E[e^{tX}] &= E\left[1+\frac{tX}{1!}+\frac{(tX)^2}{2!}+\frac{(tX)^3}{3!}+\ldots\right]\\[4pt] &= 1+\frac{t}{1!}E[X]+\frac{t^2}{2!}E[X^2]+\frac{t^3}{3!}E[X^3]+\ldots && \text{※期待値の線形性} \end{aligned} \]
期待値と無限に続く和の順序を入れ替えたり，無限に続く和を項ごとに微分したりするには，本当は収束の条件を確かめる必要があるが，ここでは気にせず進もう。

これを \(t\) で 1 回微分すると

\[ \frac{dM_X(t)}{dt} = E[X]+tE[X^2]+\frac{t^2}{2!}E[X^3]+\ldots \]

となり，\(t=0\) とすることで \(E[X]\) だけが残る：

\[ \left.\frac{dM_X(t)}{dt}\right|_{t=0} = E[X]. \]

同様に，2 回微分すると

\[ \frac{d^2M_X(t)}{dt^2} = E[X^2]+tE[X^3]+\frac{t^2}{2!}E[X^4]+\ldots \]

となり，\(t=0\) とすることで \(E[X^2]\) だけが残る：

\[ \left.\frac{d^2M_X(t)}{dt^2}\right|_{t=0} = E[X^2]. \]

3 次以上についても同様である。

ある数の並びを一つの関数にまとめ，その関数からもとの数を取り出せるようにしたものを 母関数 (generating function) という。モーメント母関数では，\(k\) 回微分して \(t=0\) とすることで，\(k\) 次の モーメント (moment) ── 確率変数の \(k\) 乗の期待値 \(E[X^k]\) ── を取り出すことができる。※

モーメント母関数を微分に使うには，\(t=0\) だけでなく，その前後の \(t\) でも \(E[e^{tX}]\) が有限である必要がある。たとえば，読み物「中心極限定理の実験 ── ベキ分布」で扱った重い裾をもつ分布のように，\(e^{tX}\) の増加が確率の減少を上回るケースでは \(E[e^{tX}]\) が有限にならない場合がある。

指数分布への応用

このモーメント母関数を用いて，指数分布の期待値と分散を求めてみよう。指数分布の確率密度関数は \(f(x; \lambda) = \lambda e^{-\lambda x}\)（\(x \geq 0\)）であった（ノート「ポアソン分布と指数分布」）。定義どおり計算すると

\[ \begin{aligned} M_X(t) = E[e^{tX}] &= \int_0^\infty e^{tx}\cdot\lambda e^{-\lambda x}\, dx = \lambda\int_0^\infty e^{(t-\lambda)x}\, dx\\[6pt] &= \lambda\left[\frac{e^{(t-\lambda)x}}{t-\lambda}\right]_0^\infty. \end{aligned} \]

定義より \(\lambda>0\)，\(t\) は最終的に 0 とするので \(\lambda>t\) とみなせば，\(e^{(t-\lambda)x}\) は \(x\to\infty\) で 0 になる。よって，

\[ M_X(t) = \frac{\lambda}{t-\lambda}(0-1) = \frac{\lambda}{\lambda-t}. \]

ここから \(t\) についての微分を計算するのは容易で，

\[ \begin{aligned} E[X] &=\left.\frac{dM_X(t)}{dt}\right|_{t=0} =\left.\frac{d\left(\lambda(\lambda-t)^{-1}\right)}{dt}\right|_{t=0} =\left.\lambda(\lambda-t)^{-2}\right|_{t=0} =\frac{1}{\lambda},\\[6pt] E[X^2] &=\left.\frac{d^2M_X(t)}{dt^2}\right|_{t=0} =\left.\frac{d\left(\lambda(\lambda-t)^{-2}\right)}{dt}\right|_{t=0} =\left.2\lambda(\lambda-t)^{-3}\right|_{t=0} =\frac{2}{\lambda^2} \end{aligned} \]

を得る。分散は \(V[X] = E[X^2] - E[X]^2\) の関係（付録「確率変数の期待値と分散の式変形いろいろ」で導出）から

\[ V[X] = \frac{2}{\lambda^2} - \left(\frac{1}{\lambda}\right)^2 = \frac{1}{\lambda^2}. \]

付録「指数分布の期待値と分散」で愚直に部分積分を繰り返した計算と比べて，ずいぶん楽になったと感じないだろうか？
