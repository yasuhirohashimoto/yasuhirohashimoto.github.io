母分散，標本分散，不偏分散

ノート「大数の法則」で，サンプルサイズ \(n\) の標本平均 \(\bar{X}_n\) は母平均 \(\mu_X\) の 一致推定量 ── \(n\) を大きくすればいくらでも真の値に近づけられる推定量 ── であることを見た．またノート「中心極限定理」では，その標本平均が母平均のまわりでどのように揺れるかを見た．本ノートではもう一度，推定量としての標本平均と分散に目を向ける．実は標本平均にはもうひとつ嬉しい性質がある．有限の \(n\) のままでも，「期待値」の意味でぴったり \(\mu_X\) と一致するのだ：

\[ \begin{aligned} E[\bar{X}_n] &= E\left[\frac{X_1 + X_2 + \cdots + X_n}{n}\right] \\[4pt] &= \frac{1}{n} \left\{E[X_1] + E[X_2] + \cdots + E[X_n]\right\} \\[4pt] &= \frac{1}{n} \left\{\mu_X + \mu_X + \cdots + \mu_X\right\}^{\text{※}} \\[4pt] &= \mu_X. \end{aligned} \]
\(X_1,\ldots,X_n\) はそれぞれが同じ分布に従う確率変数なので，それぞれの期待値 \(E[X_1],\ldots,E[X_n]\) はすべて同じ値 \(\mu_X\) になる．

「推定量の期待値」が推定したい母数とぴったり一致するとき，その推定量を 不偏推定量 (unbiased estimator) と呼ぶ．一致推定量が \(n \to \infty\) での漸近的な性質だったのに対し，不偏推定量は 有限の \(n\) でも平均的には正しい ことを保証する性質だ．つまり標本平均は，一致推定量であると同時に不偏推定量でもある，二重にありがたい推定量だったわけだ．

では，分散についてはどうだろうか．標本分散 \(S_{X,n}^2\) は，サンプルサイズを大きくすれば母分散 \(\sigma_X^2\) に確かに収束する ── つまり一致推定量である：

\[ \begin{aligned} S_{X,n}^2 &= \frac{1}{n}\sum_{i=1}^n (X_i-\bar{X}_n)^2 \\[4pt] &= \frac{1}{n}\sum_{i=1}^n \{(X_i-\mu_X)-(\bar{X}_n-\mu_X)\}^2 \\[4pt] &= \frac{1}{n}\sum_{i=1}^n (X_i-\mu_X)^2 -2(\bar{X}_n-\mu_X)\frac{1}{n}\sum_{i=1}^n (X_i-\mu_X) +(\bar{X}_n-\mu_X)^2 \\[4pt] &= \frac{1}{n}\sum_{i=1}^n (X_i-\mu_X)^2 -(\bar{X}_n-\mu_X)^2 \\[4pt] &\xrightarrow[n\to\infty]{} \sigma_X^2 - 0 = \sigma_X^2.\qquad\text{（2 つの項にそれぞれ大数の法則を適用）} \end{aligned} \]

しかし「有限の \(n\) で平均的に正しい」かどうか，つまり不偏推定量であるかどうかは別問題だ．本ノートでは結論として 標本分散は不偏推定量にはなっていない こと，すなわち \(E[S_{X,n}^2] \neq \sigma_X^2\) を示す．そのうえで，補正を施した 不偏分散 (unbiased variance) を導入する．

標本分散の期待値の計算

以下では添字を省略して，母平均を \(\mu\)，母分散を \(\sigma^2\)，サンプルサイズ \(n\) の標本平均を \(\bar{X}\)，標本分散を \(S^2\) と表記する．上で用いた式変形をそのまま用いて

\[ \begin{aligned} E[S^2] &= E\left[\frac{1}{n} \sum_{i=1}^n X_i^2 - \bar{X}^2\right] \\[4pt] &= E\left[\frac{1}{n} \sum_{i=1}^n (X_i - \mu)^2 - (\bar{X} - \mu)^2\right] \\[4pt] &= \frac{1}{n} E\left[\sum_{i=1}^n (X_i - \mu)^2\right] - E\left[(\bar{X} - \mu)^2\right] \\[4pt] &= \frac{1}{n} E\left[\sum_{i=1}^n (X_i - \mu)^2\right] - V[\bar{X}]. \end{aligned} \]

右辺第一項を展開すると，

\[ \begin{aligned} \frac{1}{n} E\left[\sum_{i=1}^n (X_i - \mu)^2\right] &= \frac{1}{n} \left\{E[(X_1 - \mu)^2] + E[(X_2 - \mu)^2] + \cdots + E[(X_n - \mu)^2]\right\} \\[4pt] &= \frac{1}{n} \left\{V[X_1] + V[X_2] + \cdots + V[X_n]\right\} \\[4pt] &= \frac{1}{n} \cdot n \sigma^2 = \sigma^2. \end{aligned} \]

右辺第二項は，標本平均の分散として

\[ \begin{aligned} V[\bar{X}] &= V\left[\frac{X_1 + X_2 + \cdots + X_n}{n}\right] \\[4pt] &= \frac{1}{n^2} \left\{V[X_1] + V[X_2] + \cdots + V[X_n]\right\} \\[4pt] &= \frac{1}{n^2} \cdot n \sigma^2 = \frac{\sigma^2}{n}. \end{aligned} \]

以上から，

\[ E[S^2] = \sigma^2 - \frac{\sigma^2}{n} = \frac{n - 1}{n} \sigma^2. \]

つまり 標本分散の期待値は母分散に一致せず，不偏推定量になっていない．具体的には \(\frac{n-1}{n}\) 倍だけ系統的に小さく出てしまう．

不偏分散

この「標本分散は平均的に母分散の \(\frac{n-1}{n}\) 倍になる」という結果を逆手にとると，標本分散を \(\frac{n}{n-1}\) 倍した値 ── つまり分散を計算する際に \(n\) ではなく \(n - 1\) で割った値

\[ s^2 = \frac{n}{n-1} S^2 = \frac{1}{n - 1} \sum_{i=1}^n (X_i - \bar{X})^2 \]

を使えば，期待値を母分散に一致させることができる．これが 不偏分散 (unbiased variance) である※．

大文字 \(S^2\) を標本分散，小文字 \(s^2\) を不偏分散と使い分けている点に注意．文献によっては記号や呼び方が違うこともあるので，前後の文脈で判別してほしい．

直感的な説明

標本分散を求めるときに引き算する標本平均 \(\bar{X}\) は，真の平均値 \(\mu\) そのものではない．真の平均値 \(\mu\) まわりの二乗偏差の期待値として定義される母分散が，目の前にある標本 \(X_i\) の中心点である \(\bar{X}\) を用いて \((X_i - \bar{X})^2\) と計算される結果，\(\mu\) を使う場合に比べて差が相対的に小さく評価されてしまう．これが 標本分散の期待値が母分散よりも小さくなる 理由である．

適当な値 \(a\) を中心とした標本の二乗偏差（ばらつき） \(\nu\) を考えたとき，この \(a\) が標本平均 \(\bar{X}\) のときに \(\nu\) は最小になることは簡単に示せる． \[ \begin{aligned} \nu &= \frac{1}{n} \sum (X_i - a)^2 = \frac{1}{n} \sum (X_i^2 - 2a X_i + a^2) \\ &= \frac{1}{n} \sum X_i^2 - \frac{2a}{n} \sum X_i + a^2. \end{aligned} \] \(\sum X_i^2\) と \(\sum X_i\) は実際の測定値から決まる定数である．したがってこの式は \(a\) についての下に凸な二次関数となり，\(a\) で微分してゼロになる点が最小値． \[ \frac{d\nu}{da} = -\frac{2}{n} \sum X_i + 2a = 0 \quad \Longrightarrow \quad a = \frac{1}{n} \sum X_i = \bar{X}. \] 目の前の標本に対する二乗偏差は，中心を \(a=\bar{X}\) に選んだとき最小になる．したがって，標本平均を中心にした標本分散 \[ \nu(a=\bar{X}) = \frac{1}{n}\sum_{i=1}^n (X_i-\bar{X})^2 \] は，真の平均 \(\mu\) を中心にした二乗偏差 \[ \nu(a=\mu) = \frac{1}{n}\sum_{i=1}^n (X_i-\mu)^2 \] よりも小さくなる方向にずれる．後者の期待値が母分散 \(\sigma^2\) なので，標本分散は平均的に母分散を過小評価してしまう．

たまにある間違い

「分散を計算するときには \(n\) ではなく \(n - 1\) で割れば偏りがないんだ！」と機械的に覚えて，どんな場面でも \(n - 1\) で割ろうとするのは誤りである．使い分けが必要だ：

  • 今，目の前にあるサンプルデータがデータのすべての場合，つまり 全数調査 (complete survey) を行う場合は，あるいは手元のデータそのもののばらつきを記述したい場合は，分散の定義として当然 \(n\) で割るのが正しい．たとえばクラスの中でテストの点数の分散を求めて，そこから個々人の偏差値を計算したいときなどはこれに該当する．
  • 一方，目の前にあるサンプルデータが巨大な母集団から抜き出された一部であり，そこから母集団全体の分散を 推定 したい場合は \(n - 1\) で割るのが自然である（平均的に偏りのない推定ができる）．たとえば同じテストについて，全国の学生の分散を，あるクラスの学生の点数から推定したい場合などはこれに該当する．

つまり，標本分散を使うべきか不偏分散を使うべきかは 分析の目的に依存する．テキストによっては単に「分散」と書いて不偏分散を指している場合があるので，前後の文脈に注意しよう※．

「\(n\) が大きければ \(n\) で割ろうが \(n - 1\) で割ろうが大して変わらないから，別に気にしなくてよいのでは？」という考え方もあるだろう．しかし \(n\) で割る操作は記述統計，\(n - 1\) で割る操作は統計的推論を想定しており，両者の態度には本質的な違いがある．
